Mitglieder
==========

Das Herzstück eines Vereins wird durch seine Mitglieder gebildet, welche zentral und mit System verwaltet werden sollten. Von goalio profitiert daher nicht nur der Vereinsvorstand, sondern auch jeder aktive Spieler, da er seine Daten selbstständig aktualisieren kann und jederzeit eine Übersicht hat, bis wann die Mitgliedsbeiträge bereits bezahlt wurden.
