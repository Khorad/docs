Mitglieder
==========

Das Herzstück eines Vereins wird durch seine Mitglieder gebildet, welche zentral und mit System verwaltet werden sollten. Von goalio profitiert daher nicht nur der Vereinsvorstand, sondern auch jeder Spieler, da er beispielsweise jederzeit den Überblick darüber hat, bis wann Mitgliedsbeiträge bezahlt wurden, bzw. in welcher Höhe der Beitrag fällig wird.
